## Useful contribution areas

- tenant-scale collection issues
- additional evidence adapters
- DuckDB read model improvements
- Entra and Azure coverage gaps
- export formats for review workflows
- anonymized ownership evidence patterns
- docs and field notes based on real tenant friction

## Contribution rule

Prefer small, reviewable changes. A narrow fix that improves collection, evidence quality or export usability is more useful than broad architecture churn.
